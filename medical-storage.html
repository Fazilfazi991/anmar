FAQ

Medical storage questions

Common questions from healthcare, supply-chain, and procurement teams.

Can medical inventory be stored with FEFO rotation?

Yes. FEFO and FIFO requirements can be discussed during quote matching so the storage workflow supports date-sensitive stock.

Which cities can medical storage support?

The site currently highlights Riyadh, Jeddah, Dammam, and Madinah coverage options.

Can you support cold-chain medical products?

If your products require chilled or frozen conditions, request cold storage matching so the warehouse profile is reviewed correctly.

What details should I send for a quote?

Send product category, pallet count or space need, storage condition, preferred city, and target start date.
